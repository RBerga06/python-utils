Welcome to :code:`rberga06-utils`'s documentation!
==================================================
:code:`rberga06-utils` is a fast and useful utility Python library, exposed under the :code:`rberga06.utils` namespace package.
It's meant to provide tools for simplifying Python application and library development and maintenance.
It's also meant to be flexible and extensible, to allow users to enhance and extend it to fit their needs.

.. note::
   This project is under active development.
..

Check out the :doc:`usage` section for further information.
